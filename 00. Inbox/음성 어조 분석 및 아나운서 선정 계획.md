# 음성 어조 분석 및 아나운서 선정 계획 (최종)

## 🎯 목표

- **극단적인 어조 값**을 가지면서 **각 어조별 분량이 충분한** 아나운서 선정
- 남자 1명, 여자 1명

---

## 분석 도구

**Parselmouth (Praat Python)** 단독 사용

- 속도: **초당 2.8ms** (가장 빠름)
- 16코어 병렬 처리로 추가 가속

---

## 분석 요소 (3가지)

|요소|추출 방법|밝음 ↔ 어두움|
|---|---|---|
|**피치 (F0)**|`sound.to_pitch()`|높음 ↔ 낮음|
|**에너지 (Intensity)**|`sound.to_intensity()`|높음 ↔ 낮음|
|**스피킹 레이트**|CSV의 `text길이 / audio_duration`|빠름 ↔ 느림|

---

## 어조 분류 기준

각 아나운서별 **개인 분포 기반** 분류:

|어조|기준 (합산 점수)|
|---|---|
|**bright**|상위 20%|
|**solemn**|하위 20%|
|**standard**|중간 30~70%|

> 경계 모호 구간 (20~~30%, 70~~80%)은 제외하여 어조 차이 극대화

---

## 실행 단계

### 1단계: Parselmouth 설치

conda activate audio_analysis

pip install praat-parselmouth

### 2단계: 샘플 테스트 (100개)

- 10개 아나운서 × 10개 파일
- 분석 속도 및 결과 검증

### 3단계: 전체 분석 (341,860개)

- 16코어 병렬 처리
- 예상 시간: ~10분

### 4단계: 아나운서별 통계

- 피치 범위 (상위 10% - 하위 10%)
- 어조별 파일 수/시간
- **선정 점수 = 피치범위 × min(bright, solemn, standard 분량)**

### 5단계: 최종 선정 및 CSV 생성

- 남성 최고 점수 1명
- 여성 최고 점수 1명
- 선정 아나운서 데이터 + tone 컬럼 추가

---

## 출력 CSV

`all_announcer_metadata.csv`에 컬럼 추가:

- `pitch_mean`: 평균 피치
- `energy_mean`: 평균 에너지
- `speaking_rate`: 스피킹 레이트
- `tone_score`: 밝음 점수 (정규화)
- `tone`: bright / standard / solemn
